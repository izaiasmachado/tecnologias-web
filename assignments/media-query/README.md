# Roteiro
## Questão 01
Faça uma DIV mudar de cor segundo a tabela abaixo. Quando a tela estiver no mínimo da largura demonstrada na tabela a cor referente a largura deverá ser empregada.

| Largura | Cor |
| ------- | --- |
| 320 | Preto |
| 480 | Azul claro |
| 600 | Verde claro |
| 768 | Amarelo claro |
| 800 | Marrom |
| 1024 | Azul escuro |
| 1280 | Verde escuro |
| 1440 | Vermelho escuro |
